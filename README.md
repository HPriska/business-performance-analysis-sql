# Business Performance Analysis (SQL)

## Deskripsi
Project ini merupakan simulasi analisis kinerja bisnis berdasarkan permintaan
berbagai stakeholder menggunakan SQL, meliputi analisis pendapatan, tren penjualan,
performa channel, funnel conversion, dan onboarding pelanggan.

## Tools
